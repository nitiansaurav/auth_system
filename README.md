# MERN Authentication System

A secure authentication system built using the MERN stack.

## Features
- User registration
- User login
- JWT authentication
- Password hashing using bcrypt
- Protected routes

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT
- bcrypt

## Installation

Clone the repository:

git clone https://github.com/nitiansaurav/auth_system.git

Install dependencies:

cd server
npm install

cd client
npm install

Run the application:

npm run dev
